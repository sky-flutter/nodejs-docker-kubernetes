# NodeJS-Docker-Kubernetes
This is sample repository which showcases how to add docker and kubernetes with the nodejs app.

### Prerequisite:
1. Kubernetes
2. Minikube
3. NodeJS

### Steps to run app
1. ``` minikube start ```
2. ``` minikube dashboard ```
3. ``` cd k8s && kubectl apply -f . ```
4. ``` minikube service node-service ```
